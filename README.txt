Project name: Mini CMU


Project description: This project creates a virtual CMU campus where the user can find the closest path to move to a building as well as see random students with different attributes. 


To run the project: 
1. Run webscraping.py (to create students.xlsx)
2. Run main 


Import:
-pandas
-beautifulsoup
-urlopen
-pil
-random
-path
